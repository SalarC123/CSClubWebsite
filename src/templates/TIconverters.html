Converters

Convert Rotations Per Minute to Radians Per Second

RPMCONRPS

This program is used for finding the radians per second of a wheel or disc when given the rotations per minute.

Convert an Orbital Meter Radius to an Astronomical Unit Radius

AUCON

This program is used for finding the orbital radius of an object in Astronomical Units (distance between Earth and Sun) when given the orbital radius in meters.

There are 3 ways to use the program:

Using the TI Connect CE Software

Download the Software at TI Connect CE Software. Locate a TI link cable or standard USB Mini Type A to USB Type Mini B. Then follow the step by step instructions after unzipping the file.

Using the TI Connect Software

Download the Software at TI Connect Software. Obtain a TI link cable or standard USB Mini Type A to USB Type Mini B. Then follow the step by step instructions after unzipping the file. For Macintosh or PC

Manually Programming Your Calculator

You can manually program the files with basic key knowledge found in this guide. You will still need to download the TI Connect or Connect CE Software to view the files.
